项目名称：音乐相册
作者：马小龙
日期：2019.7.10
功能：
双向循环链表，保存给定路径下所有的bmp和jpg文件的绝对路径。
三层界面：
             第一层：绘制封面和一个播放图标，同时监测屏幕上的点击事件，如果点击了播放图标，就进入第二层。
             第二层：后台播放一首MP3，同时显示第一张图片；
	          左滑显示下一张，右滑显示上一张；
	          上滑返回第一层，关闭音乐；
	          下滑进入第三层。
             第三层： 左滑播放下一首，右滑播放上一首；
                           上滑返回第二层；
	           下滑关闭本程序，屏幕熄灭。
